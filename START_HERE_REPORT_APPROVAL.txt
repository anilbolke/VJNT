================================================================================
   HOW TO USE - SCHOOL COORDINATOR
================================================================================

STEP 1: GENERATE REPORT
------------------------
1. Login as School Coordinator
2. Go to Dashboard
3. Click "Generate Student Report" card
4. Select a student from the list
5. Click "Generate Report"
6. Review the comprehensive report

STEP 2: SUBMIT FOR APPROVAL
----------------------------
1. In the report view, click:
   "Send to Head Master for Approval"
2. Confirm the submission
3. âœ… You will see: "Request ID: #123"
4. Status shows as "Pending Approval" (Yellow)

STEP 3: TRACK YOUR REQUESTS
----------------------------
1. Go to Dashboard
2. Click "My Report Requests" card (NEW!)
3. View all your submissions:
   - Request ID
   - Student name
   - Status (Pending/Approved/Rejected)
   - Submission date
4. Filter by status if needed

STEP 4: PRINT APPROVED REPORTS
-------------------------------
1. When Head Master approves:
   - Status changes to "Approved" (Green)
   - Print button becomes active
2. Click "View & Print Approved Report"
3. Official report opens in new tab
4. Print from there

IF REJECTED:
------------
- Status shows "Rejected" (Red)
- Rejection reason is displayed
- You can click "Resubmit for Approval"
- Make any needed changes and resubmit

================================================================================
   HOW TO USE - HEAD MASTER
================================================================================

STEP 1: VIEW PENDING REQUESTS
------------------------------
1. Login as Head Master
2. Go to Dashboard
3. Click "Approve Reports" card
4. See all pending requests with Request IDs

STEP 2: APPROVE OR REJECT
--------------------------
1. Review student data
2. Enter remarks (optional)
3. Click "Approve" or "Reject"
4. School Coordinator gets notified automatically
