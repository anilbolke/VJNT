================================================================================
                    HOW IT WORKS:
================================================================================

FLOW:
1. School Coordinator selects level in manage-students.jsp
   → Dropdown has values 0-5 (Marathi), 0-8 (Math), 0-6 (English)

2. Level number stored in database
   → marathi_akshara_level, math_akshara_level, english_akshara_level

3. When generating report:
   → Servlet reads level number from database
   → Converts number to text using switch statements
   → Returns text in JSON/HTML

4. Report displays:
   → Assessment Levels section shows the text
   → New FLN completion levels now visible
