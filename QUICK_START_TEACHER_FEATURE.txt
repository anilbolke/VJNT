================================================================================
   TEACHER ASSIGNMENT TABLE:
================================================================================

The feature requires this database table:

CREATE TABLE teacher_assignments (
    assignment_id NUMBER PRIMARY KEY,
    udise_code VARCHAR2(50),
    teacher_id NUMBER,
    teacher_name VARCHAR2(100),
    class VARCHAR2(20),
    section VARCHAR2(20),
    subjects_assigned VARCHAR2(200),  -- "Marathi,Math,English"
    is_active NUMBER(1) DEFAULT 1,
    created_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Important: subjects_assigned should contain comma-separated values:
   ‚úì Correct: "Marathi,Math,English"
   ‚úó Wrong: "marathi,math,english" (case sensitive!)

================================================================================
   DATABASE QUERY TO CHECK:
================================================================================

To verify teacher assignments in database:

SELECT ta.*, s.student_name, s.class, s.section
FROM teacher_assignments ta
JOIN students s ON ta.udise_code = s.udise_no 
    AND ta.class = s.class 
    AND ta.section = s.section
WHERE ta.is_active = 1
ORDER BY ta.udise_code, ta.class, ta.section;
